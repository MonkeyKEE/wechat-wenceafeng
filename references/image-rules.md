# 公众号文章配图与生图规则

## 一句话执行标准

> 优先使用真实、清晰、可授权且与段落强相关的网络或项目图片；真实图片不足时，再使用低AI感、无乱码、弱人物化的摄影风格AI图片补足视觉表达。

---

## 1. 配图来源优先级

**第一优先：公开网络真实图片** → **第二优先：项目自有图片** → **第三优先：AI生图**

AI生图定位：补足视觉表达，不替代真实项目记录。

---

## 2. 图片与段落匹配

每张图片必须对应当前段落的核心信息。

| 段落内容 | 推荐图片 |
|:--|:--|
| 非遗项目介绍 | 代表作品、工艺过程、工具材料、传承人工作 |
| 技艺流程 | 分步骤工艺图、手作细节、制作过程 |
| 场馆/空间 | 全景、功能区中景、局部细节、动线 |
| 活动回顾 | 人群参与、亲子互动、老师讲解、作品成果 |
| 文创产品 | 白底图、陈列图、使用场景、材质特写 |
| 品牌/情绪段 | 氛围感空间局部、材质、光影、环境 |
| 结尾总结 | 品牌空间、活动合影、代表性场景 |

---

## 3. 图片尺寸

| 类型 | 比例 | 用途 |
|:--|:--|:--|
| 常规主图 | 4:3 | 空间、展陈、活动、产品 |
| 全景图 | 3:2 / 16:9 | 场馆、舞台、商业空间 |
| 工艺/人物 | 3:4 / 4:5 | 手作、体验、人物动作 |
| 产品细节 | 1:1 / 4:5 | 文创、器物、材料局部 |
| 章节氛围 | 16:9 / 3:2 | 过渡、情绪铺垫 |
| 封面 | 2.35:1 / 16:9 | 文章头部、主视觉 |

- 默认全宽展示，不加边框/投影/拼图
- 横图为主，竖图仅作补充
- 主体不靠边，避开裁切区域

---

### 文章类型最低图片要求

| 文章类型 | 最低图片数 | 说明 |
|:--|:--:|:--|
| **活动信息释放/招募** | **不低于5张** | 如用户有拍摄素材，优先从素材库自动匹配 |
| **日常公众号分享** | **不低于8张** | 包括非遗知识、行业观察、趋势分析等 |

### 每模块建议数量

| 类型 | 建议数量 |
|:--|:--|
| 短活动预告 | 5-8张 |
| 非遗项目介绍 | 8-15张 |
| 空间探访 | 12-24张 |
| 深度场馆导览 | 18-30张 |
| 活动回顾 | 10-20张 |
| 文创产品专题 | 8-16张 |

图文节奏：短文字→主图→补充文字→连续1-2张细节图→一句总结→下一模块。

---

## 5. 图片素材分类存储

每次发布后，所有图片（网络搜索 + AI生成）统一留存到素材库，按来源分类：

```
新媒体内容创造/05_素材库/
├── 活动集锦照片/        ← 用户拍摄的项目现场图、活动图
├── AI生图照片/          ← Seedream等工具生成的AI图片
└── web搜索下载照片/     ← 网络检索下载的真实图片
```

### 存储规则

- 每次发布的图片以文章日期+主题命名子文件夹，如 `2026-06-25-暑假亲子体验/`
- 同一个子文件夹内按来源分放：`cover.png`、`scene-01.jpg` 等
- 原始网络下载图片保留原始文件名前缀，便于追溯来源
- AI生图保留生成时的prompt记录（在同目录下生成 `prompts.txt`）

---

## 5. AI生图规则

### 使用范围
非遗文化氛围图、章节过渡图、空间概念辅助图、工艺材料局部质感图、无人物的场景图、产品静物图。

### 真实感要求
真实摄影感、自然光影、低饱和、有环境细节。禁止：高饱和赛博色、过度梦幻光效、塑料感、明显AI纹理。

### 人物限制
**AI生图尽量避免以人物为主体。** 可接受：背影/侧影/远景/手部特写。禁止：面部特写、近景肖像、多人互动。

### 文字限制
**AI图片中不生成文字。** 品牌名/活动名/时间地点/价格一律不交给AI。如必须出现招牌文字，应远景化。

---

## 6. 质量控制

入库检查：与段落强相关、清晰无压缩、色调统一、无版权风险、无水印乱码、无过强AI感、适合全宽展示。

不合格即替换。
