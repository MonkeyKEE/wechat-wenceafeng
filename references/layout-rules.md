# 公众号图文排版规则

## 一句话执行标准

> 以图片承担空间叙事，以短文字完成导览解释，以留白控制阅读节奏。
> 整体：**克制、纪实、空间导览感、商业探访感**。干净，不做过度设计。

---

## 1. 整体风格

- 风格定位：克制、纪实、空间导览感、商业探访感
- 视觉主角：图片优先，文字辅助（图片占70%-85%）
- 阅读体验：像跟随作者"逛一遍空间"
- 内容节奏：短文字 + 连续图片 + 短文字 + 下一组图片

## 2. 色彩

| 用途 | 色值 |
|:--|:--|
| 标题 | #1F1F1F / #222222 |
| 正文 | #333333 / #3D3D3D |
| 底色 | #FFFFFF |

无品牌色注入。无彩色装饰。无色块背景。

## 3. 标题

| 项目 | 主标题 | 模块小标题 |
|:--|:--|:--|
| 字号 | 18-20px | 15-16px |
| 字重 | 600-700 | 600 |
| 对齐 | 左对齐 | 左对齐 |

不加色块、图标、竖条、编号。

## 4. 正文

- 字号14-15px，字重400，行距1.7-1.9
- 每段1-3行，30-80字
- 左对齐，深灰色

## 5. 图片

- 默认全宽，不加边框/投影/装饰
- 比例：全景4:3/3:2，横向16:9/3:2，细节1:1/4:5，人物4:5/3:4

## 6. 图文节奏

```
短文字 → 全景图 → 中景图 → 细节图 → 细节图 → 一句短评 → 下一模块
```

## 7. 模块分割

不用彩色分割线、花纹分隔符、装饰SVG。仅用留白+小标题。

## 8. 禁用

大段文字堆砌、高饱和色块、复杂边框花纹、大量加粗高亮、彩虹色文字、emoji、每张图配长说明、过度居中排版、强行营销口吻

## 9. 结尾

简短总结 + 地址信息 + 轻量CTA，不过度营销。
